# FlowTasks API

A simple Task Management REST API built with ASP.NET Core, Entity Framework Core, and SQL Server.

This project demonstrates real backend development skills including database integration, async APIs, and clean structure.

---

## Features
- CRUD operations for tasks
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server database
- DTOs and validation
- Async endpoints
- Swagger documentation
- Global error handling middleware

---

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- C#
- Swagger (OpenAPI)

---

## How to run the project
1. Clone the repository  
2. Make sure SQL Server Express is installed  
3. Update the connection string in `appsettings.json` if needed  
4. Run migrations:
   ```bash
   dotnet ef database update
